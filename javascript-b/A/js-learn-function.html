JavaScript Functions

Mastering Function Declarations and Anonymous Expressions

About Functions

Functions are the building blocks of JavaScript, acting as "sub-programs" that store reusable code[cite: 7, 8]. Instead of writing the same logic repeatedly, you can write it once and invoke (or "fire") it as many times as needed[cite: 11, 12].

  • Naming: Use camelCase and start with an action verb like calculateTotal()[cite: 21, 23].
  • Uniqueness: Names must be unique within their scope[cite: 17, 18].
  • Case Sensitivity: UpdateTotal() and updateTotal() are different functions[cite: 22].

1. Function Declarations

Standard Method

A function declaration uses the function keyword followed by a name[cite: 32, 33].

function displayGreeting() {
console.log('Hello, world!');
}

Key Feature: Hoisting

Function declarations are hoisted, meaning they can be called in the code even before they are defined.

2. Anonymous Functions

Function Expressions

An anonymous function has no name and is often assigned to a const variable[cite: 108, 110]. These are also known as function expressions[cite: 113].

const showGreeting = function() {
console.log('Hello from anonymous function');
};

Difference: No Hoisting

Unlike declarations, anonymous functions are not hoisted. You must define them before you try to call them.

Parameters & Arguments

Functions can accept input values to perform tasks[cite: 44].

Term Definition
Parameter The variable name defined in the function parentheses (e.g., num1)[cite: 47, 55].
Argument The actual value passed to the function when calling it (e.g., 21)[cite: 54, 56].

Pro-Tip: You can set default values to prevent errors if an argument is missing[cite: 80, 81].

function showMessage(message = "Default Error") {
console.log(message);
}

Return Values & Scope

The return Keyword

To use a result outside of a function, you must use return[cite: 67]. This also stops the function from executing any further code on subsequent lines[cite: 70, 71].

Understanding Scope

  • Global Scope: Variables defined outside functions, accessible everywhere[cite: 88, 89].
  • Local Scope: Variables defined inside a function are only accessible within that function[cite: 90].

Immediately-Invoked Anonymous Functions (IIFE)

Only anonymous functions can be run immediately without being called elsewhere[cite: 127]. This is done by wrapping the function in parentheses[cite: 128].

(function() {
console.log("I run immediately!");
})();
